# LAXPCsoft_RS
This package contains scripts that will extract the LAXPC spectrum and energy-resolved light curves. It will also correct light curves for the barycenter.

To install the package, run *"bash install-lxp.sh"*.
To extract the light curves and spectra, run *"bash laxpc_prod.sh"*.
**Note**: please set the *obsid*, *indir*, *outdir*, *ra*, and *dec*.

Download the RMF files for LAXPC10 and LAXPC20 from www.tifr.res.in/~astrosat_laxpc/LaxpcSoft.html, and keep them in the same working directory.
